# Structural (구조)

구조적 디자인 패턴은 클래스 또는 객체를 조합하여 더 큰 구조를 만들거나 새로운 구조를 만듭니다.\
변경에 유연하고 효율적으로 대처할 수 있는 구조를 설계할 수 있습니다.
